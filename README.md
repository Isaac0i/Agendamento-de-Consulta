# 🦷 Sistema de Agendamento de Consultas Odontológicas

Este projeto é uma aplicação backend desenvolvida em **Java com Spring Boot** para gerenciar o agendamento de consultas entre clientes e dentistas.

## 🚀 Funcionalidades

- Cadastro de clientes
- Cadastro de dentistas
- Definição de horários disponíveis
- Agendamento de consultas
- Listagem de consultas
- Controle de status (agendada, cancelada, concluída, etc.)
